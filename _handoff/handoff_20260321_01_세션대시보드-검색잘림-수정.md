- **substring 제거 채택** (FTS 도입, null text 개선 제외)
  - 근거: 데이터가 이미 메모리에 로드되어 있어 추가 비용 없음

  - FTS는 아키텍처 변경(self-contained HTML → 서버 또는 WASM) 필요하므로 중기 과제
